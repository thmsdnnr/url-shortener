URL Shortener Microservice

Usage:

To access a shortened URL: https://uri-shorty.herokuapp.com/[SHORTENED URL]
To enter a new URL to be shortened: https://uri-shorty.herokuapp.com/submit/[URL TO BE SHORTENED]

Note that the shortened URL must be in the correct format (beginning with http or https followed by ://www. and ending in a .(2-3 letters) suffix) to be accepted. Otherwise an error will be displayed.

Output format: JSON object {"original_url":original URL,"short_url":short URL}
